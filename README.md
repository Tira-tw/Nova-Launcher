# Nova Launcher
A lightweight, high-performance game launcher for WMMT series enthusiasts.

<img width="1280" height="1280" alt="20260626_141044-Photoroom" src="https://github.com/user-attachments/assets/352614be-e0e8-4a66-b2b0-7ee3f13cf5ce" />


## Features
- Multi-version support (W6W, WM5, W5P)
- Automated patch deployment
- MaxiTerminal orchestration
- Resolution Patcher integration

## Installation
1. `npm install`
2. `npm start`
3. `npm run dist` (to build executable)

## Note
Please run the application as Administrator.
